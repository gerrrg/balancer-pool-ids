# balancer-pool-ids
Cached poolIds for Balancer V2 pools.
Currently tracked networks:
* Ethereum mainnet
* Polygon
* Polygon zkEVM
* Arbitrum
* Optimism (Balancer ❤️  BeethovenX)
* Gnosis Chain
* Fantom (BeethovenX)
* Goerli testnet
* Sepolia testnet

## If you want to run it yourself...
### Dependencies:
Python3.9 or higher
### To run:
./getPools.sh
