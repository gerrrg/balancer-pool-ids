# balancer-pool-ids
Cached poolIds for Balancer V2 pools.
Currently tracked networks:
* Ethereum mainnet
* Kovan testnet
* Rinkeby testnet
* Goerli testnet
* Polygon
* Arbitrum
* Optimism (Balancer ❤️ BeethovenX)
* Fantom (BeethovenX)

## If you want to run it yourself...
### Dependencies:
Python3.9 or higher
### To run:
./getPools.sh
